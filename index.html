Target Trainer Help

Pick a mode (or preset), then tune settings.

Space barStarts and stops the active drill: timed board, continuous flow, or swinger.
EscHides or shows the settings panel from the main screen.

Timed Board

Timer modeThere are 3 different par timer modes: single par, reload-separated par, or shot-defined par.
Targets
  • Number of targets
  • Target shape
  • Random or grid layout
  • Optional numbered targets
Presentation
  • Random start delay arms the run
  • Hide until start beep can conceal targets
  • Start/stop or shot beeps drive the drill

Continuous Flow

Motion sourceChoose an edge, opposite-edge alternation, or border spawning.
Target stream
  • Flow speed sets travel rate
  • Creation rate sets target density
  • Random size and bright colors affect each spawned target
Run behaviorThe run starts on the start beep. Use the stopwatch for timed or target-limited runs, or bypass it to keep targets flowing until you press Stop or Space.

Swinger

Swing shapeThe selected target shape becomes one moving target.
Arc controls
  • Swing rate sets speed
  • Target height sets swing radius
  • Axis height sets pivot position
Run behaviorStart or Space begins motion. Stop or Space stops motion and returns control to the settings panel.
Screen calibration Use Calibrate Screen with a credit card to save this display's px/in. Once calibrated, known-size targets can be sized by screen distance and simulated target distance.

Screen Calibration

Overlay a credit card on the rectangle and press Confirm when the target matches the credit card size.
